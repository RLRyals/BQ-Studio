## Scene Troubleshooting

**If scene feels flat:**
- [ ] Does POV character have clear goal?
- [ ] Is there meaningful conflict?
- [ ] Does outcome matter?
- [ ] Is there emotional change?

**If scene feels slow:**
- [ ] Can it start later?
- [ ] Can it end sooner?
- [ ] Is there unnecessary description?
- [ ] Could conflict be more immediate?

**If scene feels disconnected:**
- [ ] How does it advance plot?
- [ ] How does it develop character?
- [ ] Could it be cut or combined with another scene?
