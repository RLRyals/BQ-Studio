## Character Checklist

**Before Finalizing Character:**

**Uniqueness:**
- [ ] Character feels distinct from genre archetypes
- [ ] Voice is recognizable and consistent
- [ ] Backstory creates unique perspective on investigations

**Conflict Potential:**
- [ ] Flaws create meaningful obstacles
- [ ] Relationships have tension and complexity
- [ ] Internal conflict mirrors external cases

**Investigation Capability:**
- [ ] Skills justify their position
- [ ] Weaknesses require partnerships/growth
- [ ] Abilities are useful but not overpowered

**Series Sustainability:**
- [ ] Character can grow across multiple books
- [ ] Mysteries can reasonably challenge them
- [ ] Relationships can deepen/complicate
- [ ] Powers can evolve without becoming godlike

**Reader Appeal:**
- [ ] Relatable despite extraordinary circumstances
- [ ] Likeable enough to spend series with them
- [ ] Competent but not perfect
- [ ] Active driver of plot, not passive observer

**Genre Balance:**
- [ ] Grounded enough for procedural authenticity
- [ ] Magical enough for fantasy wonder
- [ ] Professional competence balanced with personal vulnerability
- [ ] Mundane and magical elements integrated naturally
