# Case Structure Template
## Planning Individual Investigations for UFPP Books

**Genre Pack:** Urban Fantasy Police Procedural v1.0
**Usage:** Book planning phase for case-of-the-week plots
**Workflow Phase:** Book Planning - Phase 4 (Case-of-the-Week Planning)

---

## Template Instructions

Use this template to plan each book's primary investigation. Complete during book planning phase, before chapter outlining begins.

**Best Practice:** Fill out Sections 1-3 (Case Core) before proceeding to investigation beats. This ensures you know who, what, why before planning how detective discovers it.

## Section 7: Validation Checklist

**Case Logic:**
- [ ] Motive is clear and believable
- [ ] Means and opportunity are established
- [ ] Method is explained (mundane + magical)
- [ ] Timeline is coherent
- [ ] Evidence chain is logical

**Investigation Logic:**
- [ ] Detective's actions make sense given knowledge
- [ ] Deductions are supported by evidence
- [ ] False leads are plausible
- [ ] Breakthrough is earned, not convenient
- [ ] Clues are fair to reader

**Character Consistency:**
- [ ] Protagonist acts according to established personality
- [ ] Skills used match character capabilities
- [ ] Emotional responses are authentic
- [ ] Growth is organic to events
- [ ] Knowledge state is tracked

**Procedural Authenticity:**
- [ ] Police procedures are realistic (or adapted reasonably)
- [ ] Legal requirements are addressed
- [ ] Departmental politics are considered
- [ ] Timelines are believable

**Genre Balance:**
- [ ] Mundane and magical elements are integrated
- [ ] Urban fantasy voice is maintained
- [ ] Procedural beats are present
- [ ] World-building is natural, not info-dumpy
- [ ] Case feels grounded despite magic

**Series Integration:**
- [ ] Standalone case is satisfying
- [ ] Series arc is advanced
- [ ] Character relationships progress
- [ ] World is expanded appropriately
- [ ] Hook for next book is present (if not final book)
